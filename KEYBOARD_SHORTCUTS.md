| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd+K` / `Ctrl+K` | Command Palette | Quick navigation and actions |

## Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `J` | Next Item | Navigate down in lists (vim-style) |
| `K` | Previous Item | Navigate up in lists (vim-style) |
| `G` then `G` | Go to Top | Jump to first item |
| `Shift+G` | Go to Bottom | Jump to last item |

## Data Interaction

| Shortcut | Action | Description |

## Tips

1. **Keyboard-First**: All critical actions are keyboard accessible
2. **Vim-Style**: J/K navigation for power users
3. **Monospace Values**: All auditable data uses monospace font
4. **Copy-Friendly**: Click any hash or ID to copy
5. **Provenance**: Press `?` on any element to see its history
