What does "Learn More" do?

The user clicks "Learn More" on the Keon Collective option during integration selection.

A

Full-page navigation within the wizard

The wizard navigates to a showcase page (e.g., /setup?step=collective-showcase). The showcase renders inside the same onboarding shell with a ← Back button that returns them to the split screen. No context is lost.

B

Modal / slide-over on top of the split screen

The showcase content slides in as an overlay or modal on top of the integration selection step. The user closes it and is back exactly where they were — split screen intact, selection state preserved.

C

Opens in a new tab

The showcase opens in a new browser tab so the wizard stays exactly where it is. Good for users who want to read without losing their place, but adds friction to return and continue.
